## Descripción

Este repositorio contiene el desarrollo del Taller No. 3 de Análisis Estadístico Descriptivo y Visualización de Datos.

El objetivo del trabajo fue analizar la variable **Apps Descargadas** a partir de una muestra de 50 observaciones, aplicando técnicas de estadística descriptiva para organizar, resumir e interpretar los datos mediante tablas de frecuencia y representaciones gráficas.

## Contenido del Repositorio

- Informe final del taller.
- Archivo de datos utilizado para el análisis.
- Tabla de distribución de frecuencias.
- Histograma de frecuencias.
- Polígono de frecuencias.
- Ojiva o curva acumulada.

## Resumen del Análisis

Durante el desarrollo del taller se calcularon el rango, el número de clases mediante la regla de Sturges y la amplitud de clase. Posteriormente se construyó la tabla de distribución de frecuencias y se elaboraron los gráficos estadísticos correspondientes.

Los resultados mostraron una mayor concentración de observaciones en los intervalos centrales, especialmente entre 60 y 75 aplicaciones descargadas. Además, se determinó que el 80 % de los usuarios había descargado hasta 105 aplicaciones.
