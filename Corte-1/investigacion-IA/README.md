# Contenido del trabajo

El notebook contiene las siguientes secciones:

## 1. Algoritmo A*

Se presenta una explicacion teorica del algoritmo de busqueda A*, uno de los algoritmos mas utilizados para encontrar rutas optimas en grafos.

Se explica el funcionamiento de la funcion:

f(n) = g(n) + h(n)

donde:

- g(n) representa el costo desde el nodo inicial hasta el nodo actual
- h(n) representa una estimacion del costo restante hasta el objetivo (heuristica)

Tambien se incluye una implementacion del algoritmo en Python que muestra el recorrido encontrado.

---

## 2. Heuristica

Se explica el concepto de heuristica y su importancia en los algoritmos de busqueda.

Se abordan los siguientes puntos:

- Que es una heuristica
- Para que sirve
- Como funciona

Ademas se presentan ejemplos practicos como:

- Sistemas de navegacion GPS
- Evaluacion de posiciones en juegos como el ajedrez

---

## 3. Teoria de Juegos

Se introduce el concepto de teoria de juegos como una herramienta para analizar decisiones estrategicas entre multiples agentes.

Se explican los siguientes aspectos:

- Que es la teoria de juegos
- Para que sirve
- Como funciona

Tambien se presentan tres ejemplos:

1. Dilema del prisionero  
2. Competencia entre empresas  
3. Subastas  

Estos ejemplos muestran como las decisiones de un participante pueden influir en las decisiones y resultados de los demas.
